#脚本简介

##说明
主要是针对百度的SEO脚本，提供查询收录，排名等多个功能；因为只是函数，所以还需要引用才能使用。

##功能及函数

*  siter(url) - 查询百度收录site数值，应直接输入要查询的网址，如"www.tuniu.com";
*  indexer(url) - 查询百度是否收录，查询最终URL，100%准确；
*  simple_indexer(url) - 查询百度收录，简单版；
*  ranker(word, url) - 查询百度排名，查询最终URL，100%准确；
*  simple_rankder(word, url) - 简单版百度排名查询，查询的是快照URL。
